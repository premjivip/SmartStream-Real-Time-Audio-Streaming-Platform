                                     SMART STREAM - REAL-TIME-AUDIO-STREAMING-PLATFORM

                                                       
                Project Overview:  

                    Smart Stream is a real-time Audio streaming web application developed using Spring Boot.
                    It allows users to register, login, upload Audio, and watch song securely using JWT authentication.

                Features:
                
                  * User Registration & Login
                  * JWT-based Authentication
                  * Audio Upload & Streaming
                  * Search and Browse Audio
                  * Secure REST APIs
                  * User Profile Management

               Technologies Used:

                  * Java
                  * Spring Boot
                  * Spring Security
                  * JWT (JSON Web Token)
                  * Hibernate / JPA
                  * MySQL
                  * Thymelef

               How to Run: 

                  * Clone the repository:
                  * git clone https://github.com/your-username/smart-stream.git
                  * Open in IDE (Eclipse / IntelliJ)
                  * Configure database in application.properties
               
              Run the project:
                  * mvn spring-boot
                  * Open in browser:
                  * http://localhost:8080

              Project Structure:

                  * controller → Handles API requests
                  * service → Business logic
                  * repository → Database operations
                  * model → Entity classes

             Authentication Flow:

                 * User logs in with username & password
                 * Server validates credentials
                 * JWT token is generated
                 * Token is used to access secured APIs
